Welcome to the URL Shortener project! This repository contains a simple URL shortening service that takes long URLs and generates short, unique aliases for them. This README file provides an overview of the project, instructions for setting it up, and details on how to use the URL shortener.

# Features :

The URL Shortener comes with the following features:

- Generate short aliases for long URLs.
- Redirect users to the original long URL when they visit the shortened URL.
- Statistics tracking for the number of times each shortened URL has been visited.
- Simple and intuitive web interface for creating and managing shortened URLs.
